# Rust Solar

A library for cosmic calendars. 
 
## How to Install?

```sh
cargo add rust solar -F asteroids planets moons exoplanets comets
```

## Note
 * This library is compatible with leptos as it's used in [SuperZone](https://superzone.vercel.app/)
